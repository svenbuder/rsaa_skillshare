# RSAA Skillshare (2026-04): GitHub Basics and Workflows Using Online Presence as an Example

**Audience:** Anyone at RSAA
**Format:** 15 minute intro + 60 minute interactive workshop on using git to create your website

## Material

- [Slides (PDF)](rsaa-skillshare-2026-04-git-slides.pdf)
- [Tutorial (PDF)](rsaa-skillshare-2026-04-git-tutorial.pdf)

## Session goals

Participants should leave with:
- an idea of how reproducible research is achieved
- exposure to `git` as one of the providers for version control
- a personal website hosted at `username.github.io` that was created with active and development branches, pull-requests, and an action workflow

## In case you want to adjust slides

You need `pandoc` and `xelatex` as well as the files in `src/`. From within `src/`, then simply execute `make` in `bash`.
